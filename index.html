Pico WiFi Duck

About Pico WiFi Duck

Turn your Raspberry Pi Pico W into a USB Rubber Ducky. Run Ducky Script remotely via Wi-Fi and simulate keyboard inputs on a connected system.

How to Use

  • Connect to hotspot: PicoWiFiDuck | Password: !@picowsecurity
  • Write or upload your .txt Ducky Script.
  • Click Run to launch the payload.
  • Sample Script: 
    REM Open Notepad and say hi
GUI r
DELAY 300
STRING notepad
ENTER
DELAY 500
STRING Hello from Pico WiFi Duck!
